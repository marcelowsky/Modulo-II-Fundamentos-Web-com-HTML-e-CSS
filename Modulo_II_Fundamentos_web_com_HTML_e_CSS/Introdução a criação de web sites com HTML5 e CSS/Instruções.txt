Termo Chaves
*Download/upload
*Cache: arquivos salvos no computador quando se acessa um site
*host: servidor
*HTML: linguagem de marcação da internet (não faz funções)
*HTTP:Protocolo da Internet
*Link: é uma ligação
*Url: Endereço de um site
*Keylogger: Software malicioso que registra ações do teclado
*Phishing: Software malicioso para roubar dados atraves de e-mail e msg
*Bullying: Ameaças, difamação através de meio eletrônico
*Cookies: Registros no computador utilizados geralmente em ofertas
*Click bait: Publicações/msg o qual a intenção é que o usuário "clique"

REDES
*Backbone: estruturas que gerenciam as conexões da internet e redes
*Provedor de acesso:Empresas que vendem acessos aos Backbones
*Provedor de Serviço: Fibra ótica - Rádio - Satélite - Móvel
*www - DNS - IP - Classes de redes - 127.0.0.1 Local host

TCP/IP: Protocolos de comunicação entre computadores
* 4 Camadas:
1.Física (placa de rede)
2.Rede (IP)
3.Transporte (TCP/UDP)
4. Aplicação (FTP,SMTP, HTTP)

UDP: Rápido/não confiável/"Carro do Ovo"/Livestream
TCP: Voltado a conexão/Handshake/Integridade/App msg texto

PORTAS:Identificação das conexões e qual o tipo de informação trafegará (22/23/25/443/ etc)

MODEMS: Modula e demodula sinais.
ROTEADOR: Distribuidor de sinais para um ou mais dispositivos.
SWITCHES: Distribuidor de sinais "inteligente"
Portas utilizada pelo Gmail: IMAP
